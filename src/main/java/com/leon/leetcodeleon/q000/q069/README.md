```
实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

示例 1:

输入: 4
输出: 2
示例 2:

输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/sqrtx
```

思路
=
因为这里不需要小数，那么就可以使用整数的猜法，再配合二分法，使计算加快。

由于一个数的平方根不会超过它的一半，上边界可以直接使用 x/2,以加快运算
